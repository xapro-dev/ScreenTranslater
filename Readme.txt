Description
Reads screen area, retrieves text and translates it.

Build
pyinstaller -Dyw --hidden-import=pyttsx3.drivers --hidden-import=pyttsx3.drivers.sapi5 main.py

Installation
Install tesseract from https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-v4.1.0.20190314.exe
Extract archive anywhere and run main.exe !from administrator!.
Works only with Windows 10.
Enjoy

Usage
Press "SELECT" and select area with text.
Press "Ahide" to enable autohide after 20 seconds after translation.
Press "mute" to disable Text-To-Speech.
Press "run" to work. lol.
